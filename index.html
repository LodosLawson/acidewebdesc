AC-T-IDE
Azerbaycan Programlama Dili IDE

Azerbaycan dilinde programlama yapmak için geliştirilmiş entegre geliştirme ortamı

Version 5.0 2024 Multi-Platform
Kurulum Kılavuzu Kodları İncele

Genel Bilgi

AC-T-IDE, Azerbaycan dilinde programlama yapmak için özel olarak tasarlanmış bir entegre geliştirme ortamıdır. Desktop, Android ve Web platformlarında çalışabilen çok platformlu bir IDE'dir.

Geliştirme Tarihi

Proje 2022 yılında başlatılmış ve sürekli geliştirilmektedir. İlk versiyon (V0.3) 14 Haziran 2022'de yayınlanmıştır.

Geliştirici

Mehemmed AC tarafından geliştirilmiştir. Proje açık kaynak kodlu olup GitHub üzerinden yayınlanabilir.

Kurulum Talimatları

Desktop IDE Kurulumu

1

Gereksinimler

  • Java JDK 13 veya üzeri
  • Maven 3.6+
  • NetBeans IDE (opsiyonel)
2

Projeyi İndirin

git clone [repository-url]
cd ACTKernal
3

Maven ile Derleyin

mvn clean install
4

Çalıştırın

java -jar target/AC-IDE-AC-T-IDE\ Version\ 3.0.jar

Android IDE Kurulumu

1

Gereksinimler

  • Android Studio
  • Android SDK 24+
  • Gradle 8.1+
2

Projeyi Açın

Android Studio'da AndroidVersion/ACIDEAndroid klasörünü açın
3

Firebase Yapılandırması

google-services.json dosyasını app/ klasörüne ekleyin

4

Derleyin ve Çalıştırın

./gradlew assembleDebug
# veya Android Studio'dan Run butonuna basın

Python Interpreter (KBIR) Kurulumu

1

Gereksinimler

  • Python 3.7+
2

KBIR Klasörüne Gidin

cd AndroidVersion/ACIDEAndroid/KBIR
3

Çalıştırın

python main.py
